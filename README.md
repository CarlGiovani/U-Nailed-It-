U NAILed It by Alliyah

 - - - - - 

ABOUT

A booking and management website for U NAILed It by Alliyah, a nail salon business. This project handles appointments, services, and admin dashboard.

 - - - - - 

OVERVIEW

This system is designed to help customers book nail services efficiently and allow the salon to manage schedules, clients, and service availability.

 - - - - - 

PROPOSED FEATURES

- Online booking system
- Service list with prices
- Appointment management and database
- Admin dashboard
- Automatic time conflict prevention
- Push Notifications
